# Book Tracker
Мобильное приложение для чтения книг и их учета для формирования привычки чтения.
Настроила локальную базу данных для хранения информации о книгах и статистики, реализовала отображение статистики в виде календаря и гистограммы, а также возможность загрузки книги и её чтение в приложении.
